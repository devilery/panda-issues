@xeger @taylor-cedar @zan-rosenthal @ashanbrown @ccpost @majormoses @rouge8 This is released in closed beta and I've enabled it for your accounts. You can enable it from "Your DM Settings". Note that you'll need to install an additional GitHub App to enable the webhooks needed for this. Also, this feature does NOT use GitHub's new checks API. Please let me know your feedback! Things I'm looking for feedback on:

  • Is the overall functionality and format of the Slack message good?
  • For now I have the substrings circleci, travis-ci, and codeship whitelisted so that any failed check containing one of those substrings in its name will trigger a Slack DM. Let me know if you'd prefer to be able to control which checks are whitelisted via substring match or regex.
